Plugin Activation

At first upload the plugin.Then you find a slider menu on your dashboard.It shows 3 sub menu

  1. all slider image
  2. add slider image
  3. slider categor

 

Create Slider

First click slider category then you find a input box for add your slider categor.Exp: i need a slider for my home page.

So my slider categoy is home.After create slider category you find some field for customize the plugin like as auto,effect etc.Select a effect for your sider otherwise it will be normal effect default.In slider category page look on the url(top) bar,exp:

wp-admin/edit-tags.php?action=edit&taxonomy=slider-category&tag_ID=43&post_type=slider

Remember this id,it's need for shortcode.

 

Then add your image for slider with selecting your sider category.

Use Shortcode

Suppose i want to use slider on my home page.Then open home page(edit) using dashboard.Think my slider category id is 43,the i have use shortcode [slider id="43"]

Very simple and easy.
